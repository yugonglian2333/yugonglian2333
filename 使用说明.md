# 使用说明

这个压缩包已经把朋友仓库中的内容改成了适合你的 GitHub Profile README 版本。

## 1. 仓库名称

GitHub 主页仓库的名字必须和你的 GitHub 用户名完全一致。

如果你的 GitHub 用户名是：

```txt
bryceandrewsm74
```

那么仓库名就应该是：

```txt
bryceandrewsm74
```

并且仓库需要设置为 Public。

## 2. 上传文件

把本文件夹中的这些内容上传到你的同名仓库根目录：

```txt
README.md
README.zh.md
.github/workflows/snake.yml
assets/
```

## 3. 开启 Actions 写入权限

进入你的仓库：

```txt
Settings → Actions → General → Workflow permissions
```

选择：

```txt
Read and write permissions
```

然后保存。

## 4. 运行贪吃蛇动画

进入：

```txt
Actions → Generate Contribution Snake → Run workflow
```

运行成功后，`assets/` 里的 SVG 会被替换成你自己的贡献图贪吃蛇动画。
